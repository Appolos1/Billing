Installation

# Install CLI globally cd billing/cli npm install npm install -g . --force # Verify installation billing --help

Start the proxy

# With API key from dashboard (recommended) billing start --key bill_sk_your_key_here # Or authenticate with email first billing login billing start

Connect your AI tool

# Claude Code ANTHROPIC_BASE_URL=http://127.0.0.1:9099 claude # Cursor — Settings → Models → Base URL http://127.0.0.1:9099 # Cline / RooCode — API Base URL setting http://127.0.0.1:9099 # Windsurf — Cascade → API endpoint http://127.0.0.1:9099

All CLI commands

billing login # sign in to your account billing start --key bill_sk_... # start proxy with API key billing start # start proxy with session token billing stop # stop the proxy billing status # show proxy status billing whoami # show current user billing keys list # list your API keys billing keys create "Name" # create a new API key billing skill upgrade # AI skill upgrade (+XP) billing logout # sign out

Environment variables (server)

# .env or shell exports ANTHROPIC_API_KEY=sk-ant-... # Anthropic key OPENROUTER_API_KEY=sk-or-... # OpenRouter key (free models) REQUIRE_APPROVAL=false # disable waitlist (default: true) HOURLY_LIMIT=200 # requests per hour per user DAILY_LIMIT=1000 # requests per day per user PORT=3000 # server port

Data collection notice

Billing collects prompts and completions for benchmarking and AI model training. All data is anonymized. By using this service you agree that your prompts may be used to improve AI models. No personally identifiable information is stored alongside prompt data.
